# Arete

A frameless, always-on-top desktop widget for staying on top of your day, built with Electron. Everything is stored locally — no account, no cloud, fully yours.

The name comes from the Greek *aretē* (ἀρετή) — excellence, and the act of striving to be the best version of yourself. That is what this little widget is for.

## Features
- **Tasks** — add (Enter), check off, delete, double-click to edit. Clickable HIGH / MED / LOW priority tags and per-task due dates (Today / Tomorrow / Overdue chips).
- **Filters** — All / Active / Done / High priority, plus Clear done.
- **Habits** — track daily habits with 🔥 streaks and a rolling 7-day grid.
- **Stats** — a GitHub-style contribution graph of your activity, with current streak, best streak, and weekly / monthly completion counts.
- **7 themes** — Glassmorphism, Waterdrop, Neomorphism, Neo Brutalism, Neo Brutal Soft, Brutalism, and a cream + pink Brand theme. Plus accent colours and an opacity slider.
- **Collapse mode** — ▾ shrinks the widget down to just the list so it tucks neatly into a corner of your screen.
- **Always-on-top** — 📌 pins it above every other window. Drag the top bar to move, drag the edges to resize. Position, size, pin state, opacity, and all your data are remembered between launches.

## Install
```
npm install
```

## Run
```
npm start
```
Or double-click `Tasks Widget.vbs` on Windows to launch it silently (no console window).
